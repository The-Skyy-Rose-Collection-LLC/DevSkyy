# SkyyRose Visual Reference Set — Canonical

**Locked by founder 2026-05-25.** Source of truth for any visual / aesthetic / design work. Every mockup, brand pitch, agent brief, landing page, collection page, social asset, and brand-touching artifact begins with this reference list.

## The Five

| # | Brand | What we pull from them |
|---|-------|------------------------|
| 1 | **Kith** ([kith.com](https://kith.com)) | Monogram-led editorial photography. Magazine-grid product layouts. Heritage anchoring. Retail-store integration. Cinematic lookbooks. |
| 2 | **Oaklandish** ([oaklandish.com](https://oaklandish.com)) | Civic pride messaging baked into the design. Tree icon as logo (parallel to SR monogram). "For The Town" tagline. Local-landmark photography. Accessible luxury price-point that still feels collected. |
| 3 | **Culture Kings** ([culturekings.com](https://culturekings.com)) | Dense product display on drops. Hip-hop / music culture integration. High-energy editorial videos. Sneaker hero presentation. Drop-moment density. |
| 4 | **Fear of God** ([fearofgod.com](https://fearofgod.com)) | Cinematic photography (sepia/desaturated). Oversized type on hero. Sport DNA (basketball-derived silhouettes). Spiritual/elevated copy. Family/family-name imagery. |
| 5 | **Palm Angels** ([palmangels.com](https://palmangels.com)) | Sport heritage motifs (racing stripes, athletic logos). Embroidery details extreme close-up. Vintage athletic photography colorism. |

## Shared DNA across the set

- **Photography-led**, not graphics-led — real people in real places
- **Monogram/logo as protagonist**, not as afterthought
- **Heritage anchoring** — the city or culture is part of the brand identity
- **Magazine-grid product layouts** — editorial spread, not retail catalog
- **Disciplined palettes** with selective brand-color drops
- **Athletic / sport DNA** underneath the luxury polish

## Why this set, specifically, fits SkyyRose

| Reference move | SkyyRose application |
|----------------|----------------------|
| Kith's monogram-led editorial | SR rose-gold monogram as protagonist in every hero |
| Oaklandish's civic pride | "The Town" anchoring, "FOR THE TOWN" copy, Lake Merritt / Bay Bridge / Port Of Oakland imagery |
| FOG's cinematic photography | Urban Oakland industrial settings as photographic environment |
| Palm Angels' sport heritage | NFL / NBA / MLB / Hockey patches across the Black Rose collection |
| Culture Kings' drop density | Launch moment editorial spreads, sport-patch lookbook density |

## Locked-out references (do not use)

These were referenced in prior brand/design work and produced visuals that did not fit. Do not pull from this list when designing for SkyyRose:

- Bottega Veneta
- Numéro / Document Journal
- Hedi Slimane Celine
- Rick Owens (visual register only; garment-construction references still OK if specifically relevant)
- Acne Studios FW24
- Givenchy by Tisci
- 032c
- Off-White (early era, Demna pre-2024)
- Burberry Imagined
- Apple Vision Pro as an aesthetic reference (the device itself is fine to reference as a product)

These references pulled the visual direction toward European luxury house lineage — museum / vault / cathedral aesthetic. That is the wrong lineage for SkyyRose. SkyyRose is American luxury streetwear with Oakland civic-pride and Black-owned heritage. The Five above provide the correct visual grammar.

## Adjacent (pull only with justification)

These brands are stylistically near the right register and can be referenced when the brief specifically needs what they bring:

- **Telfar** — when Black-owned, accessible, gender-neutral overlap is the point
- **Pyer Moss** — when Black-owned cultural narrative is the point
- **Aimé Leon Dore** — when retail-store-as-brand-anchor is the point
- **Rhude** — when cultural-grounding via athletic luxe is the point

## Practical use

When opening any visual brief — mockup, lookbook spread, agent prompt, landing page design — start with these five. The first sentence of your visual reasoning should name which of the five you're pulling from and what specific move you're adapting. If none of the five fit, say so explicitly and explain why before reaching for an adjacent reference.
