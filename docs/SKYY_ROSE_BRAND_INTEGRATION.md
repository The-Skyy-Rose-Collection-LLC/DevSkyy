## Content Guidelines

### Always Include
1. "The Skyy Rose Collection" brand name
2. Premium quality and craftsmanship messaging
3. Sustainability and ethical production references
4. Customer-centric value propositions
5. Call-to-action with urgency (limited edition, exclusive)

### Never Include
1. Discounts over 30% (maintains luxury positioning)
2. Comparisons to fast fashion brands
3. Mass-market or budget terminology
4. Overly technical AI/tech jargon
5. Generic stock imagery without brand watermark

### Tone Examples

**✅ Good:**
> "Discover timeless elegance with The Skyy Rose Collection's new arrivals. Crafted from Italian silk and designed for the modern woman who values sophistication."

**❌ Bad:**
> "Check out our cheap dresses on sale now! Limited time only!"

### Quick Start

1. **Load Brand Config:**
```python
from agents.mcp.orchestrator import MCPOrchestrator

orchestrator = MCPOrchestrator(
    config_path="/tmp/DevSkyy/config/mcp/mcp_tool_calling_schema.json",
    brand_config_path="/tmp/DevSkyy/config/mcp/skyy_rose_brand_config.json"
)
```

2. **All Tasks Auto-Validate Against Brand:**
```python
task = await orchestrator.create_task(
    name="Generate Instagram post for new dress",
    agent_role=AgentRole.VISUAL_FOUNDRY,
    tool_name="stable_diffusion",
    input_data={
        "prompt": "Elegant woman wearing rose gold dress, Skyy blue background",
        "style": "luxury fashion editorial"
    }
)

# Orchestrator automatically validates:
# - Brand colors present
# - Luxury aesthetic
# - Photography style compliance
# - Logo watermark requirement

result = await orchestrator.execute_task(task)
```

3. **Brand KPI Tracking:**
