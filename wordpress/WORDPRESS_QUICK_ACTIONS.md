# WordPress Quick Actions (5 Minutes)

**Login**: https://skyyrose.co/wp-admin

## Task 1: Activate Spinning Logo Plugin
1. **Plugins** → Find "SkyyRose 3D Experience" → **Activate**

## Task 2: Apply Custom CSS
1. **Appearance → Customize → Additional CSS**
2. Open `~/Desktop/skyyrose-luxury.css` in TextEdit
3. Copy all 785 lines → Paste → **Publish**

## Task 3: Configure Menu
1. **Appearance → Menus** → Create "Primary Navigation"
2. Add pages in order:
   - Home
   - Collections (with BLACK ROSE, LOVE HURTS, SIGNATURE as sub-items)
   - Blog
   - About
3. Check "Primary Menu" location → **Save Menu**

---

**Done!** Your site will have luxury styling, working navigation, and spinning logo.
