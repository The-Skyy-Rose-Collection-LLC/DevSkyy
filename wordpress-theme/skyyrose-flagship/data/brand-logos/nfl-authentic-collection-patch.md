# NFL Authentic Collection Patch — Canonical Reference

The NFL-themed Authentic Collection patch used on the Black Rose football
jersey series (br-008 SF Inspired, br-009 Last Oakland). Mirrors the MLB-style
"Black Rose Authentic Collection" patch (see
`black-rose-authentic-collection-patch.md`) but swaps the MLB batter
silhouette for the NFL shield.

## Visual Description

A roughly rectangular / portrait-oriented patch with two halves divided by a
thin red diamond divider strip:

### Upper half — the AUTHENTIC card

- **Outer border**: thin double frame — outer line in dark navy, then a hairline
  white inset, surrounding…
- **Yellow / mustard-gold field** (the dominant background color of the upper
  card).
- **"BLACK ROSE"** — large dark-navy serif text arched across the top of the
  yellow card.
- Below the text, the **NFL shield logo**: a vertical-elongated shield with
  navy blue field, red and white "NFL" lettering centered, white stars
  arranged in an arc above the lettering, and a small white-and-brown football
  illustration above the stars.
- A **white scroll banner** crosses below the shield with the word
  **"AUTHENTIC"** in dark navy text, edges of the banner accented in red.
- Below the banner, **"COLLECTION"** in large dark-navy serif text, mirroring
  the "BLACK ROSE" text at the top.

### Divider

- A horizontal strip of **red diamonds** (small diamond shapes in a row,
  roughly 9–10 visible, dark red on white).

### Lower half — the place tag

- **White / cream background**.
- **"Property Of"** in italic dark-serif text on the left.
- A **greyscale Black Rose three-rose-cluster** illustration centered (the
  canonical rose-cluster art with the cloud at base — see
  `black-rose-logo.md`).
- **"Black Is Beautiful Series"** in bright red script across the lower-mid
  area.
- **"Skyy Rose Co"** in italic dark-serif text along the bottom-right.

## What this patch IS NOT

- NOT the MLB Authentic Collection patch (that's
  `data/brand-logos/black-rose-authentic-collection-patch.md` — uses a batter
  silhouette and "Members Only / Yay Area, Ca / Made for Kings ang Queens"
  footer copy).
- NOT the NBA Authentic Collection patch (different sport silhouette).
- NOT the Hockey Championship patch (greyscale palette, different layout).

## How techniques apply this patch

This patch is typically rendered as a **woven or embroidered patch** sewn
onto the football jersey. The intricate text + NFL shield + multi-color frame
mean it is rarely embossed or printed flat — woven gives the cleanest
legibility.

- **embroidered-patch**: Patch backing fabric with embroidered text + figures.
  Most common application on jerseys.
- **woven-label**: Smaller hangtag-style applications.
- **printed / sublimated**: Possible for fabric-back applications, retains
  full multi-color fidelity.
