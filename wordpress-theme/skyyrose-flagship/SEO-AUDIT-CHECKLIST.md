# SEO Audit Checklist

Comprehensive SEO optimization checklist for the SkyyRose Flagship Theme.

## On-Page SEO

### Title Tags
- [ ] Every page has a unique title
- [ ] Titles are under 60 characters
- [ ] Primary keyword is in the title
- [ ] Title accurately describes page content
- [ ] Brand name is included (except homepage)
- [ ] No duplicate titles across site

### Meta Descriptions
- [ ] Every page has a meta description
- [ ] Descriptions are 150-160 characters
- [ ] Descriptions include target keywords
- [ ] Descriptions are compelling and actionable
- [ ] No duplicate descriptions across site

### Headings
- [ ] Only one H1 per page
- [ ] H1 contains primary keyword
- [ ] Heading hierarchy is logical (H1 → H2 → H3)
- [ ] Headings describe content accurately
- [ ] No heading levels are skipped

### URLs
- [ ] URLs are descriptive and readable
- [ ] URLs use hyphens (not underscores)
- [ ] URLs are lowercase
- [ ] URLs are short and concise
- [ ] URLs include target keywords
- [ ] No dynamic parameters when possible

### Content
- [ ] Content is original and valuable
- [ ] Primary keyword appears naturally
- [ ] Content is well-structured with headings
- [ ] Paragraphs are short and scannable
- [ ] Images have descriptive alt text
- [ ] Internal links use descriptive anchor text
- [ ] External links open in new tab
- [ ] Content length is adequate (300+ words)

### Images
- [ ] All images have alt text
- [ ] File names are descriptive
- [ ] Images are compressed/optimized
- [ ] Responsive images are used
- [ ] Important images use lazy loading
- [ ] Product images have Schema markup

### Internal Linking
- [ ] Related products are linked
- [ ] Blog posts link to relevant products
- [ ] Footer links to important pages
- [ ] Breadcrumbs are implemented
- [ ] No broken internal links

## Technical SEO

### Crawlability
- [ ] robots.txt is accessible
- [ ] robots.txt doesn't block important pages
- [ ] XML sitemap is generated
- [ ] XML sitemap is submitted to Search Console
- [ ] All important pages are in sitemap
- [ ] No sitemap errors in Search Console

### Indexability
- [ ] Important pages are indexable
- [ ] Duplicate content has canonical tags
- [ ] Staging/dev sites use noindex
- [ ] Pagination uses rel="next/prev"
- [ ] Archive pages have proper robots tags

### Site Speed
- [ ] PageSpeed Insights score is 90+
- [ ] First Contentful Paint < 1.8s
- [ ] Largest Contentful Paint < 2.5s
- [ ] Time to Interactive < 3.8s
- [ ] Cumulative Layout Shift < 0.1
- [ ] Images are optimized
- [ ] CSS is minified
- [ ] JavaScript is minified
- [ ] Browser caching is enabled

### Mobile Optimization
- [ ] Site is mobile-responsive
- [ ] Mobile-friendly test passes
- [ ] No mobile-specific errors in Search Console
- [ ] Touch targets are 48x48px minimum
- [ ] Text is readable without zooming
- [ ] No horizontal scrolling

### HTTPS & Security
- [ ] Site uses HTTPS
- [ ] SSL certificate is valid
- [ ] All resources load over HTTPS
- [ ] No mixed content warnings
- [ ] Security headers are set

### Canonical URLs
- [ ] Every page has a canonical tag
- [ ] Canonical points to correct version
- [ ] HTTPS canonical for HTTPS sites
- [ ] No chains of canonical tags

## Schema Markup

### General Schema
- [ ] Organization schema on homepage
- [ ] Logo is defined in schema
- [ ] Social media profiles in schema
- [ ] Contact information in schema
- [ ] BreadcrumbList on all pages
- [ ] WebSite schema with search

### Product Schema
- [ ] Product schema on all product pages
- [ ] Name, description, image included
- [ ] Price and currency specified
- [ ] Availability status set
- [ ] SKU included
- [ ] Brand included
- [ ] Reviews/ratings included
- [ ] Rich results test passes

### Review Schema
- [ ] AggregateRating on products
- [ ] Individual reviews marked up
- [ ] Rating values are correct
- [ ] Review count is accurate

### Other Schema
- [ ] FAQ schema where applicable
- [ ] Article schema for blog posts
- [ ] CollectionPage for categories
- [ ] Event schema where applicable

## Social Media

### Open Graph
- [ ] og:type is set correctly
- [ ] og:title matches page title
- [ ] og:description is compelling
- [ ] og:image is high-quality (1200x630px)
- [ ] og:url is canonical URL
- [ ] og:site_name is set
- [ ] Product-specific OG tags for products

### Twitter Cards
- [ ] twitter:card type is set
- [ ] twitter:site handle is set
- [ ] twitter:title is set
- [ ] twitter:description is set
- [ ] twitter:image is optimized
- [ ] Cards validate in Twitter validator

### Social Sharing
- [ ] Share buttons are present
- [ ] Shares preview correctly
- [ ] Facebook debugger shows correct info
- [ ] Twitter card validator passes

## WooCommerce SEO

### Product Pages

### Category Pages
