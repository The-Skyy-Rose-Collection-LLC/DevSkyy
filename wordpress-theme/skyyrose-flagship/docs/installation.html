Installation

Follow these steps to install the SkyyRose theme and get your store running.

System Requirements

SoftwareMinimum Version
WordPress6.8+
PHP8.2+
WooCommerce9.9+
MySQL / MariaDB8.0+ / 10.6+
PHP Memory Limit256 MB recommended

Step 1 — Upload the Theme

  1. In your WordPress dashboard, go to Appearance > Themes > Add New > Upload Theme.
  2. Click Choose File and select the skyyrose-flagship.zip file from your download.
  3. Click Install Now and wait for the upload to complete.
  4. Click Activate.

Step 2 — Install WooCommerce

  1. Go to Plugins > Add New and search for WooCommerce.
  2. Install and activate WooCommerce (version 9.9 or higher is required).
  3. Complete the WooCommerce setup wizard, or skip it and configure settings manually later.
If WooCommerce is already installed, verify your version at Plugins > Installed Plugins. Update if below 9.9.

Step 3 — Import Demo Content via Blueprints

  1. After activation, go to Appearance > SkyyRose Setup.
  2. Click Import Demo Content. This uses WC Blueprints for one-click import.
  3. The importer will create: sample products, collection pages, landing pages, menus, and Customizer settings.
  4. Wait for the progress bar to complete. Do not navigate away during import.

Step 4 (Optional) — Install Elementor Pro

  1. Purchase and download Elementor Pro from elementor.com.
  2. Go to Plugins > Add New > Upload Plugin and install the Elementor Pro zip.
  3. Activate both Elementor and Elementor Pro.
  4. Go to Elementor > Template Kit Library and import the SkyyRose Template Kit included in your download (the skyyrose-template-kit.zip file).
Elementor Pro is optional. SkyyRose works fully with Gutenberg, Divi, Beaver Builder, and Bricks as well.

Step 5 — Verify Installation

After completing the steps above, confirm everything is working:

  • Visit your homepage — you should see the SkyyRose front page layout.
  • Check WooCommerce > Status for any warnings.
  • Visit Appearance > Customize to confirm theme options are available.
