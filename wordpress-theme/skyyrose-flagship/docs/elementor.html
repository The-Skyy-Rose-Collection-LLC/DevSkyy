Elementor Pro Integration

SkyyRose is built Elementor Pro native. A full Template Kit is included with pre-designed templates for every theme builder location.

Importing the Template Kit

  1. Ensure both Elementor (free) and Elementor Pro are installed and activated.
  2. Go to Elementor > Template Kit Library.
  3. Click Import Template Kit and select the skyyrose-template-kit.zip file from your theme download.
  4. Choose which templates to import (recommended: import all).
  5. Click Import. Templates are now available in Templates > Saved Templates.

Dynamic Tags

SkyyRose registers custom Elementor dynamic tags you can use in any widget:

  • SkyyRose Collection Name — Outputs the name of the current collection based on the page or product category. Use in headings, text widgets, or breadcrumbs.
  • KC Status — Outputs the current Kids Capsule status ("Launch Mode" or "Live"). Useful for conditional display logic or admin-facing labels.

To insert a dynamic tag: click the dynamic tag icon (stacked database icon) in any text field within the Elementor editor, then search for "SkyyRose".

Page Templates

SkyyRose provides two Elementor-specific page templates:

  • Elementor Canvas — A completely blank canvas with no header, footer, or sidebars. Use for landing pages, coming-soon pages, or immersive experiences. Template file: template-elementor-canvas.php.
  • Elementor Full Width — Full-width layout that keeps the theme header and footer but removes sidebars and content width constraints. Template file: template-elementor-fullwidth.php.

Assign these templates under Page Settings > Page Layout in the Elementor editor, or via Page Attributes > Template in the WordPress editor.

Disabling Google Fonts

SkyyRose uses self-hosted fonts via the WordPress Font Library. To prevent Elementor from loading Google Fonts:

  1. Go to Elementor > Settings > Advanced.
  2. Set Google Fonts Load to Disable.
  3. Save changes.

This ensures zero external font requests, maintaining full GDPR compliance and optimal performance.

Elementor Pro is optional. All SkyyRose features work without it using the block editor, block patterns, and PHP templates. See Page Builders for other builder options.
