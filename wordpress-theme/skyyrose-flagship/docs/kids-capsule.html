Kids Capsule

The Kids Capsule collection has two distinct modes: Launch Mode (pre-release teaser) and Live Mode (full catalog). Toggle between them from the Customizer.

Launch Mode vs. Live Mode

Launch Mode

  • Teaser landing page with brand imagery
  • Countdown timer to launch date
  • Email waitlist signup form
  • Products are hidden from the shop
  • Collection page redirects to the teaser

Live Mode

  • Full product catalog visible
  • Collection page displays all Kids Capsule products
  • Parent-child matching commerce enabled
  • Immersive 3D experience accessible
  • All block patterns available

Toggling Modes

  1. Go to Appearance > Customize > Kids Capsule Launch.
  2. Set Collection Status to either Launch Mode or Live Mode.
  3. When in Launch Mode, configure the following fields:
    • Launch Date — The date and time for the countdown timer.
    • Teaser Heading — Main heading displayed on the teaser page.
    • Teaser Description — Short paragraph below the heading.
    • Waitlist Form Action — URL for your email service (Mailchimp, Klaviyo, etc.).
    • Teaser Background Image — Hero image for the teaser page.
  4. Click Publish to save.

Launch Mode Details

Teaser Page

When Launch Mode is active, visitors to any Kids Capsule URL see a full-screen teaser page with:

  • A hero background image with overlay
  • Your custom heading and description
  • An animated countdown timer showing days, hours, minutes, and seconds
  • An email signup form for the waitlist

Countdown Timer

The countdown automatically calculates time remaining from the current date to your configured launch date. When the date passes, it displays "Available Now" but does not automatically switch to Live Mode — you must toggle manually in the Customizer.

Email Waitlist

The waitlist form submits to your configured form action URL. It collects the visitor's email address and sends it to your email marketing service. The form includes honeypot spam protection and WCAG-compliant labeling.

Parent-Child Matching Commerce

In Live Mode, the Kids Capsule supports parent-child matching — allowing customers to buy matching adult and kids items together:

  • Products linked via the Matching Parent Product field in the product editor.
  • A "Shop the Match" button appears on both the adult and kids product pages.
  • Add-to-cart for matching items creates a bundled line item in the WooCommerce Store API cart.
  • Matching products display side-by-side on the collection page with a visual connector.
When switching from Launch Mode to Live Mode, clear any page caches to ensure visitors see the full catalog immediately.
