WooCommerce Setup

SkyyRose requires WooCommerce 9.9+ and leverages the Store API v1 for a modern, AJAX-powered shopping experience.

Requirements

  • WooCommerce 9.9 or newer
  • WordPress 6.7+
  • PHP 8.2+
  • SSL certificate (required for checkout)

Creating Products

  1. Navigate to Products → Add New.
  2. Enter the product name, description, and pricing in the Product Data panel.
  3. Upload a product image at the recommended size of 1200 × 1500 px. AVIF format is supported for smaller file sizes.
  4. Add gallery images for the holographic card hover effect (minimum 2 images recommended).
  5. Assign the product to a Collection category (see below).

Store API v1 Cart

SkyyRose replaces the traditional WooCommerce AJAX cart with the Store API v1 endpoint for faster, headless-compatible cart operations.

// AJAX add-to-cart via Store API v1
fetch('/wp-json/wc/store/v1/cart/add-item', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Nonce': skyyrose_cart.nonce
  },
  body: JSON.stringify({ id: productId, quantity: 1 })
});

The mini-cart drawer updates automatically after each Store API response. No page reload is required.

Blueprints Demo Import

SkyyRose ships with a WooCommerce Blueprints file for one-click demo content import:

  1. Go to WooCommerce → Home.
  2. Click Import Blueprint when prompted, or navigate to Tools → Import.
  3. Select the blueprint.json file from the theme's /assets/blueprints/ directory.
  4. The importer creates sample products, collections, pages, and menus.

Custom Product Meta Fields

Two custom meta fields are added to the Product Data panel:

FieldMeta KeyDescription
Care Instructions_skyyrose_careWashing, drying, and ironing guidelines displayed on the product page.
Fabric Content_skyyrose_fabricMaterial composition (e.g., “100% Premium Cotton”) shown in the product details tab.

These fields appear in a dedicated SkyyRose tab within the Product Data metabox.

Pre-Order Gateway

SkyyRose includes a lightweight pre-order payment gateway for upcoming drops:

  • Enable at WooCommerce → Settings → Payments → SkyyRose Pre-Order.
  • Mark individual products as pre-order in the Product Data panel.
  • Customers are charged when the product ships (manual capture) or immediately (auto-capture).
  • Pre-order badges display automatically on product cards and the cart.

Checkout Flow

  1. Cart Drawer — Slide-out mini-cart powered by Store API v1. Quantity adjustments happen inline.
  2. Checkout Page — Uses the WooCommerce Checkout block for a streamlined, single-page checkout.
  3. Order Confirmation — Branded thank-you page with order summary, social sharing, and next-drop teaser.

The entire flow uses GSAP micro-animations for transitions, keeping the luxury feel consistent through purchase completion.
