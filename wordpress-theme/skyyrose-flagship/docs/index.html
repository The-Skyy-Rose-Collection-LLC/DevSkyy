SkyyRose Theme Documentation

Luxury Grows from Concrete.

SkyyRose is a luxury streetwear WooCommerce theme built for bold brands. It ships with four curated collections, immersive Three.js experiences, GSAP-powered animations, and full compatibility with every major WordPress page builder.

Feature Highlights

  • 4 Collections — Black Rose, Love Hurts, Signature, and Kids Capsule, each with dedicated landing pages, catalog pages, and 3D immersive experiences.
  • WooCommerce 9.9+ — Store API cart, product quick-view, and one-click demo import via WC Blueprints.
  • Elementor Pro Native — Template Kit with pre-built header, footer, single product, and archive templates.
  • Universal Builder Support — Works with Divi, Beaver Builder, Bricks, and the block editor (Gutenberg).
  • Three.js & GSAP — Holographic product cards, glassmorphism navigation, and scroll-driven animations.
  • WCAG 2.2 AA Accessible — Focus management, skip links, contrast ratios, and reduced-motion support.
  • Self-Hosted Fonts — Uses the WordPress Font Library. Zero Google Fonts CDN calls for full GDPR compliance.
  • Kids Capsule Launch Mode — Built-in teaser page with countdown timer and email waitlist.
  • Block Patterns — Pre-designed patterns for every collection, ready to insert from the block editor.
