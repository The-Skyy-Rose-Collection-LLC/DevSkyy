Changelog

All notable changes to the SkyyRose theme are documented below. This log follows Keep a Changelog conventions.

Version Format

SkyyRose uses semantic versioning: MAJOR.MINOR.PATCH

  • MAJOR — Breaking changes (template structure, required plugin versions).
  • MINOR — New features and enhancements (backward-compatible).
  • PATCH — Bug fixes, security patches, performance improvements.

Upgrade Instructions

Before updating, always back up your site and any child theme customizations. After uploading the new version:

  1. Navigate to Appearance → Themes and activate the updated theme.
  2. Go to Settings → Permalinks and click Save to flush rewrite rules.
  3. Clear any page or object caches.
  4. Review the notes below for any version-specific steps.

1.0.0

— April 12, 2026

Initial Release

  • Feature Four luxury collections: Black Rose, Love Hurts, Signature, and Kids Capsule.
  • Feature WooCommerce 9.9+ integration with Store API v1 AJAX cart.
  • Feature Elementor Pro native support with custom widgets for collections and holographic product cards.
  • Feature Divi 5.x, Beaver Builder 2.x, Bricks 1.x, and Gutenberg (Site Editor) compatibility.
  • Feature Block patterns for hero banners, product grids, collection showcases, and testimonial rows.
  • Feature WooCommerce Blueprints demo import for one-click sample content setup.
  • Feature Self-hosted font system with 9 families (Inter, Playfair Display, Cinzel, Cormorant Garamond, Bebas Neue, DM Sans, Barlow, Oswald, Instrument Serif) via Font Library.
  • Feature Three.js holographic product cards and GSAP scroll animations.
  • Feature AVIF image support with custom image sizes (hero, product, card, thumbnail).
  • Feature theme.json design tokens: 15 colors, 6 gradients, fluid typography with clamp().
  • Feature Pre-order payment gateway for upcoming drops.
  • Feature Custom product meta fields for care instructions and fabric content.
  • Feature Canvas and Full Width page templates for all supported builders.
  • Feature Kids Capsule toggle in the Customizer for site-wide visibility control.
  • Feature CSS custom properties (--skyyrose-*) for easy child theme overrides.
