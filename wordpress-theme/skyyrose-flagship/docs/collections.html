Collections

SkyyRose ships with four curated collections. Each collection includes a landing page, a product catalog page, and an immersive Three.js experience. Block patterns are provided for all collections.

General Setup

  1. Create a WooCommerce product category matching the collection name (done automatically during demo import).
  2. Assign products to the appropriate category.
  3. Each collection has a dedicated page template for its landing page and immersive experience. Assign these under Page Attributes > Template.
  4. Insert collection-specific block patterns via Block Editor > Patterns > SkyyRose.

Block Patterns

Each collection ships with pre-designed block patterns you can insert from the editor:

  • Collection Hero — Full-width hero with background image, heading, and CTA button.
  • Product Showcase — 3-column product grid with holographic card effect.
  • Lookbook Strip — Horizontal scrolling image gallery.
  • Story Section — Two-column text + image layout for brand storytelling.

Access patterns via Block Editor toolbar (+) > Patterns > SkyyRose [Collection Name].
