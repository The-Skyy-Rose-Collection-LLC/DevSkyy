Getting Started

This guide walks you through your first-launch configuration: branding, menus, homepage, and collection pages.

1. Customizer Settings

Go to Appearance > Customize to access all SkyyRose theme options.

Brand Colors

  • Accent Color — defaults to Rose Gold #B76E79. Used for buttons, links, and highlights.
  • Background Color — defaults to #0A0A0A. The dark base of the theme.
  • Text Color — defaults to #F5F5F0.

Logo

  • Navigate to Site Identity in the Customizer.
  • Upload your logo (recommended size: 200x60px, SVG or PNG with transparency).
  • Set the site title and tagline as fallback text.

Social URLs

  • Navigate to SkyyRose Settings > Social Links.
  • Enter URLs for Instagram, TikTok, X (Twitter), Facebook, and Pinterest.
  • Icons will appear automatically in the header and footer.

2. Setting Up Menus

Go to Appearance > Menus (or use the Customizer under Menus).

SkyyRose registers four menu locations:

  • Primary — Main navigation in the header. Add your collection pages, Shop, About, and Contact.
  • Footer Shop — Appears in the footer. Suggested links: Shop All, New Arrivals, Sale, Gift Cards.
  • Footer Help — Appears in the footer. Suggested links: FAQ, Shipping & Returns, Size Guide, Contact.
  • Footer Legal — Appears in the footer. Suggested links: Privacy Policy, Terms of Service, Cookie Policy.

Create a menu for each location and assign it under Menu Settings > Display location.

3. Configuring the Homepage

  1. Go to Settings > Reading.
  2. Select A static page and choose the Home page (created during demo import).
  3. Set the Posts page to your blog page if you plan to use one.
  4. The homepage template includes: hero section, featured collections grid, lookbook carousel, and newsletter signup.
If you did not import demo content, create a new page and assign the Front Page template under Page Attributes > Template.

4. Setting Up Collection Pages

Each collection has three page types, created automatically during demo import:

  1. Landing Page — The marketing entry point with hero imagery and storytelling. Uses a dedicated page template (e.g., template-landing-black-rose.php).
  2. Collection Page — The WooCommerce product archive filtered to that collection's category.
  3. Immersive Experience — A Three.js-powered interactive page. Uses a dedicated template (e.g., template-immersive-black-rose.php).

Link these pages in your Primary menu under each collection's dropdown.

For detailed collection setup, see Collections.
