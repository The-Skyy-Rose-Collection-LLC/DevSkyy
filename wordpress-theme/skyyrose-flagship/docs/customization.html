Customization

SkyyRose exposes design tokens through theme.json, the WordPress Customizer, CSS custom properties, and the Font Library. Every visual aspect of the theme can be adjusted without editing PHP.

Fluid Typography

Font sizes use clamp() for responsive scaling without breakpoints:

"fontSize": "clamp(1rem, 0.5rem + 1vw, 1.125rem)"  /* Body */
"fontSize": "clamp(2rem, 1rem + 3vw, 4rem)"          /* Hero heading */

Custom Image Sizes

SkyyRose registers four image sizes on activation. Upload full-resolution images and WordPress will generate each crop:

NameDimensionsUsage
skyyrose-hero1920 × 1080Full-width hero banners
skyyrose-product1200 × 1500Single product page main image
skyyrose-card600 × 750Product grid cards, holographic hover
skyyrose-thumb400 × 500Thumbnails, related products

AVIF format is supported. Upload .avif files directly or use a plugin to auto-convert on upload.

CSS Custom Properties

All theme tokens are exposed as CSS variables on :root for use in custom CSS or child themes:

:root {
  --skyyrose-color-base: #0A0A0A;
  --skyyrose-color-contrast: #F5F5F0;
  --skyyrose-color-accent: #B76E79;
  --skyyrose-color-black-rose: #C0C0C0;
  --skyyrose-color-love-hurts: #DC143C;
  --skyyrose-color-signature: #D4AF37;
  --skyyrose-color-kids-capsule: #B76E79;
  --skyyrose-font-body: 'Inter', system-ui, sans-serif;
  --skyyrose-font-heading: 'Playfair Display', serif;
  --skyyrose-radius-sm: 4px;
  --skyyrose-radius-md: 8px;
  --skyyrose-radius-lg: 16px;
  --skyyrose-shadow-card: 0 4px 24px rgba(0,0,0,0.4);
}

Override any property in a child theme's style.css or via Appearance → Customize → Additional CSS.
