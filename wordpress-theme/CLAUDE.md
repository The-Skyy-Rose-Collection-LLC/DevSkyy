# WordPress Theme

> Production theme: `skyyrose-flagship/` (v3.2.0) -- the ONLY theme in this directory.

## Key Rules

- **REST API**: Use `index.php?rest_route=` NOT `/wp-json/` (WordPress.com requirement)
- **Immersive pages** = 3D storytelling (NOT shopping)
- **Catalog pages** = product grids (FOR shopping)
- **No `skyyrose-2025`** -- that theme does not exist, only `skyyrose-flagship`
