  <action>
Create tests/test_color_contrast_wcag.py. The file has three sections:

1. Inline WCAG helper (~25 lines, no external deps beyond stdlib):
   - srgb_linearize(c: int) -> float: applies WCAG 2.x gamma formula. If c/255 <= 0.04045 return (c/255)/12.92 else return ((c/255 + 0.055)/1.055)^2.4.
   - relative_luminance(hex_color: str) -> float: strips "#", splits into R/G/B ints, applies srgb_linearize, returns 0.2126*R + 0.7152*G + 0.0722*B.
   - wcag_ratio(hex_a: str, hex_b: str) -> float: computes (L_lighter + 0.05) / (L_darker + 0.05).
   - alpha_blend(r: int, g: int, b: int, alpha: float, bg_hex: str) -> str: blends rgba foreground on solid bg. For each channel: out = round(alpha * fg_ch + (1 - alpha) * bg_ch). Returns "#RRGGBB" hex.

2. Token parser (module-level constant):
   - TOKENS_CSS_PATH = Path(__file__).parent.parent / "wordpress-theme/skyyrose-flagship/assets/css/design-tokens.css"
   - parse_root_tokens() -> dict[str, str]: reads file, finds ":root {" block, extracts lines matching "--skyyrose-(text|text-muted|bg|accent|secondary)\s*:\s*([^;]+);" using re.findall. Returns dict of name->raw_value. Stops at closing "}" after ":root {".
   - parse_collection_tokens() -> dict[str, dict[str, str]]: finds each "[data-collection=...]" block, extracts same token names per slug. Returns dict of slug->overrides (only tokens actually present in that block).
   - effective_tokens(slug: str | None) -> dict[str, str]: merges root tokens with collection overrides. For slug=None, returns root.

3. Tests (pytest functions, no class):
   - test_wcag_helper_known_pair: assert wcag_ratio("#F5E6D3", "#0A0A0A") >= 15.0. (Known ~17:1 — floor set conservatively to tolerate floating-point rounding.)
   - test_wcag_ratio_order_independent: assert abs(wcag_ratio("#F5E6D3", "#0A0A0A") - wcag_ratio("#0A0A0A", "#F5E6D3")) < 0.01.
   - test_text_muted_blend: parse root tokens, extract rgba values (r=245,g=230,b=211,alpha=0.7), blend on bg (#0A0A0A), assert wcag_ratio(blended, "#0A0A0A") >= 4.5.
   - test_root_text_bg_meets_wcag_aa: parse root tokens, assert wcag_ratio("--skyyrose-text value", "--skyyrose-bg value") >= 4.5.
   - test_all_collections_inherit_root_text_bg: for each slug in ["black-rose", "love-hurts", "signature", "kids-capsule"], call effective_tokens(slug), confirm neither "--skyyrose-text" nor "--skyyrose-bg" is overridden (i.e., value equals root), assert ratio >= 4.5.
   - test_tokens_css_exists: assert TOKENS_CSS_PATH.exists(), message includes path.

   Parsing notes:
   - text-muted raw value is "rgba(245, 230, 211, 0.7)". Parser must extract r/g/b/a from this format using regex: r"rgba\(\s*(\d+)\s*,\s*(\d+)\s*,\s*(\d+)\s*,\s*([\d.]+)\s*\)".
   - Hex values may have leading/trailing whitespace — strip before use.
   - Do NOT import from skyyrose.core (pure CSS parse, zero project dep).
