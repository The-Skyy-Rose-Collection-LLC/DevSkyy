    - Test: static tokens are monotonically increasing in rem value (xs < sm < base < lg < xl < 2xl)
    - Test: TOKENS_CSS_PATH.exists() — fast guard that fails early if CSS file moves
    - Test: 320px regression — no inline style= in fixture contains width:\s*(\d+)px where int > 320 AND the same style attribute lacks overflow:(auto|hidden|scroll)
  </behavior>
  <action>
Create tests/test_responsive_tokens.py. The file has three sections:

1. Path constants (module-level):
   TOKENS_CSS_PATH = Path(__file__).parent.parent /
     "wordpress-theme/skyyrose-flagship/assets/css/design-tokens.css"
   FIXTURE_PATH = Path(__file__).parent / "fixtures/homepage_skyyrose.html"

2. CSS parsers (pure stdlib, no external deps):

   parse_clamp_tokens() -> dict[str, tuple[str, str, str]]:
     Reads TOKENS_CSS_PATH. Finds all lines matching:
       r"(--text-[a-z0-9-]+)\s*:\s*clamp\(([^)]+)\)"
     For each match, splits the inner string on "," to get [min, preferred, max].
     Strips whitespace from each part. Returns dict of name -> (min, preferred, max).
     Skips tokens whose value does not start with "clamp(" (handles static-rem tokens).

   parse_static_tokens() -> dict[str, float]:
     Reads TOKENS_CSS_PATH. Finds all lines matching:
       r"(--text-[a-z0-9-]+)\s*:\s*([\d.]+)rem\s*;"
     Returns dict of name -> float (rem value). Excludes any token whose line
     contains "clamp(" (those are handled by parse_clamp_tokens).

   rem_to_float(rem_str: str) -> float:
     Strips trailing "rem", returns float. Example: "1.875rem" -> 1.875.

3. Tests (pytest functions, no class):

   test_tokens_css_exists:
     assert TOKENS_CSS_PATH.exists(), f"design-tokens.css not found at {TOKENS_CSS_PATH}"

   test_clamp_token_count:
     tokens = parse_clamp_tokens()
     assert len(tokens) >= 7, f"Expected >=7 clamp tokens, got {len(tokens)}: {list(tokens)}"
     expected = {"--text-3xl", "--text-4xl", "--text-5xl",
                 "--text-decorative-sm", "--text-decorative-md",
                 "--text-decorative-lg", "--text-decorative-xl"}
     assert expected.issubset(tokens.keys()), f"Missing clamp tokens: {expected - tokens.keys()}"

   test_clamp_tokens_three_args:
     for name, (mn, pref, mx) in parse_clamp_tokens().items():
         assert mn and pref and mx, f"{name}: clamp() has empty argument"

   test_clamp_min_floor:
     for name, (mn, _, _) in parse_clamp_tokens().items():
         assert rem_to_float(mn) >= 0.75, f"{name}: min {mn} is below 0.75rem floor"

   test_clamp_preferred_uses_vw:
     for name, (_, pref, _) in parse_clamp_tokens().items():
         assert "vw" in pref, f"{name}: preferred arg '{pref}' does not use vw unit"

   test_clamp_max_gte_min:
     for name, (mn, _, mx) in parse_clamp_tokens().items():
         assert rem_to_float(mx) >= rem_to_float(mn), \
             f"{name}: max {mx} < min {mn}"

   test_static_token_count:
     tokens = parse_static_tokens()
     expected = {"--text-xs", "--text-sm", "--text-base",
                 "--text-lg", "--text-xl", "--text-2xl"}
     assert expected.issubset(tokens.keys()), f"Missing static tokens: {expected - tokens.keys()}"

   test_static_tokens_monotonic:
     tokens = parse_static_tokens()
     order = ["--text-xs", "--text-sm", "--text-base",
              "--text-lg", "--text-xl", "--text-2xl"]
     values = [tokens[k] for k in order if k in tokens]
     for i in range(len(values) - 1):
         assert values[i] < values[i + 1], \
             f"Static token scale is not monotonic at index {i}: {values[i]} >= {values[i+1]}"

   test_320px_no_overflow_violations:
     Heuristic: parse fixture with re.findall to extract all style="..." attribute values.
     For each style value:
       - Search for r"width\s*:\s*(\d+)px" — extract int width.
       - If width > 320, check whether the same style value contains
         r"overflow\s*:\s*(auto|hidden|scroll)" (case-insensitive).
       - If NOT, record violation as f"width:{width}px without overflow handling".
     If FIXTURE_PATH does not exist: pytest.skip("fixture not available").
     Assert len(violations) == 0, f"320px overflow violations: {violations}".

   Implementation note: Use re, not BeautifulSoup — zero extra deps beyond stdlib.
   Do NOT import from skyyrose.core (pure CSS/HTML parse, zero project dep).
  </action>
