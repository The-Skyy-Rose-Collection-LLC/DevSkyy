    - A11Y-09: The first `&lt;img&gt;` in the page body has `loading="eager"` (or no loading attribute, which is eager by default). All subsequent `&lt;img&gt;` elements that appear after the first have `loading="lazy"`. Test against homepage fixture only (hero image is above fold).
