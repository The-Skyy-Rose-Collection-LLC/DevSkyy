# The Five

Canonical visual references SkyyRose pulls from:

1. **Kith** — luxury streetwear, layered storytelling, lookbook spreads, founder-driven
2. **Oaklandish** — Oakland authenticity, civic pride, community-rooted, founder-driven
3. **Culture Kings** — motion-heavy hero, drop culture, urgency without cheapness
4. **Fear of God** — editorial restraint, monochrome palette discipline, fashion-cathedral
5. **Palm Angels** — graphic identity, sport+luxury fusion, capsule drops

**Why:** Founder locked this set on 2026-05-25 after multiple corrections away from European luxury house references.

**How to apply:** every visual / aesthetic / mood reference cites one of these five by specific example ("Kith's Fall 2024 lookbook hero" — not "luxury homepages"). NEVER cite: Bottega Veneta, Numéro, Hedi Slimane, Rick Owens, 032c, Acne FW24, Givenchy by Tisci, Khaite, Bode, The Row.
