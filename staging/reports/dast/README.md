# DAST Reports Directory

This directory contains vulnerability scan reports from OWASP ZAP and Nuclei.

## Directory Structure

```
dast/
├── zap_report_YYYYMMDD_HHMMSS.json          # Raw ZAP findings
├── zap_report_YYYYMMDD_HHMMSS.html          # ZAP visual report
├── nuclei_report_YYYYMMDD_HHMMSS.json       # Raw Nuclei findings
├── zap_parsed_YYYYMMDD_HHMMSS.json          # Parsed ZAP findings
├── nuclei_parsed_YYYYMMDD_HHMMSS.json       # Parsed Nuclei findings
├── combined_vulnerability_report_*.json      # Unified findings
├── vulnerability_summary_*.txt               # Human-readable summary
├── vulnerability_delta_*.json                # Baseline comparison
├── vulnerability_baseline.json               # Current baseline
└── archive/                                  # Old reports (30+ days)
```

## Report Types

### Raw Reports
- **zap_report_*.json**: OWASP ZAP alert data in native format
- **zap_report_*.html**: Visual HTML report from ZAP
- **nuclei_report_*.json**: Nuclei findings in JSONL format

### Parsed Reports
- **zap_parsed_*.json**: Normalized ZAP findings with OWASP mapping
- **nuclei_parsed_*.json**: Normalized Nuclei findings with CVE/CWE data

### Combined Reports
- **combined_vulnerability_report_*.json**: Deduplicated findings from all sources
- **vulnerability_summary_*.txt**: Executive summary
- **vulnerability_delta_*.json**: Changes from baseline

### Baseline
- **vulnerability_baseline.json**: Known vulnerabilities for tracking
- **vulnerability_baseline.example.json**: Example baseline structure

## Viewing Reports

### Quick Summary
```bash
cat vulnerability_summary_*.txt | tail -50
```

### Check for Blockers
```bash
jq '.blockers | length' combined_vulnerability_report_*.json
```

### List Critical Issues
```bash
jq '.vulnerabilities[] | select(.severity == "critical")' combined_vulnerability_report_*.json
```

### View Statistics
```bash
jq '.statistics' combined_vulnerability_report_*.json
```

### Compare with Baseline
```bash
jq '.statistics | {new_count, fixed_count, net_change}' vulnerability_delta_*.json
```

## Report Retention

- **Active Reports**: Keep for 30 days
- **Archive**: Automatically moved after 30 days
- **Baseline**: Keep indefinitely, update after verified fixes

## Report Generation

Reports are automatically generated by:
```bash
./staging/run_dast_scan.sh
```

## Security

- Reports may contain sensitive information
- Restrict access to authorized personnel
- Do not commit reports to version control
- Use `.gitignore` to exclude reports
