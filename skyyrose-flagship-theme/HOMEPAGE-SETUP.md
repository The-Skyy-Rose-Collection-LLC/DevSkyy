# Set Up Custom Homepage - Quick Guide

## WordPress.com Dashboard Steps

### Step 1: Create/Edit Homepage
Navigate to: **Pages → All Pages**

Option A: Edit existing "Home" page
- Click: "Home" page
- Page Attributes → Template: "Custom Homepage (Full Design)"
- Click: Update

Option B: Create new homepage
- Click: Add New
- Title: "Home"
- Page Attributes → Template: "Custom Homepage (Full Design)"
- Content: (Leave blank - template handles everything)
- Click: Publish

### Step 2: Set as Homepage
Navigate to: **Settings → Reading**

Configure:
- ☐ Your homepage displays: **A static page**
- ☐ Homepage: **Home** (select the page you just created/edited)
- ☐ Posts page: (Optional - select "Blog" if you have one)

Click: **Save Changes**

### Step 3: Verify
1. Visit: https://skyyrose.co
2. Should see:
   - Custom navigation with collection links
   - Luxury brand styling (rose gold/gold)
   - "SKYYROSE" logo in header
   - Collection links: Black Rose | Love Hurts | Signature

### Step 4: Clear Cache
Navigate to: **Hosting Configuration → Performance**
- Click: **Clear Cache**
- Wait: 60 seconds

## Troubleshooting

**Issue: Homepage shows generic layout**
- Fix: Verify template selected: "Custom Homepage (Full Design)"
- Location: Page edit screen → Page Attributes → Template

**Issue: Navigation links 404**
- Fix: Create product categories first (Phase 3, Step 1)
- Categories needed: black-rose, love-hurts, signature-collection

**Issue: Changes don't appear**
- Fix: Clear WordPress cache
- Location: Hosting Configuration → Performance → Clear Cache
